# 🚆 Train Delay Prediction using Machine Learning

## 📌 Overview
This project predicts train arrival delays using machine learning techniques based on operational and environmental data.

---

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Flask
- Docker

---

## 📊 Features
- Data preprocessing pipeline
- Random Forest model
- Model evaluation (MAE, R²)
- REST API for predictions
- Docker containerization
